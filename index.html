项目概述

基于论文《文旅融合视角下皖北军事环境成因及数字传播方法研究》构建的知识图谱可视化平台

📚

学术研究

为历史地理学领域的学术研究者提供便捷、高效的数字工具，帮助理解事件之间的时空关系与人物参与的历史背景

🔍

数据可视化

结合知识图谱与地理地图，实现历史事件、人物、地点的多维度可视化展示与交互分析

🌐

数字传播

支持数据上传、查询、可视化分析等功能，可嵌入至展览、数据库门户或文旅系统中

核心功能

强大的功能模块，满足历史数据研究的多方面需求

📊

知识图谱可视化

  • 使用 Cytoscape.js 实现图谱节点与连线的交互显示
  • 悬停查看属性，点击展开详情
  • 支持节点筛选与关系探索
  • 可导出图谱子集为图片
🗺️

地理地图展示

  • 使用 Leaflet 显示事件的地理分布
  • 支持坐标聚合与筛选
  • 基于时间/人物/地点的多维度筛选
  • 历史地图图层切换功能
📤

数据上传管理

  • 支持 CSV / JSON 格式上传
  • 自动解析字段并检查格式正确性
  • 管理员上传后将数据写入数据库
  • 显示上传记录与构建状态
🔎

智能查询筛选

  • 支持关键词检索（事件名、人名、地点、时间范围）
  • 条件组合查询（如：1940年前、某人、某地）
  • 查询结果可在图谱与地图模块中同步高亮
  • 支持模糊匹配与精确查询
⏱️

时间轴视图

  • 时间维度的事件展示
  • 支持时间范围筛选
  • 事件发展链条演绎
  • 历史时间线可视化
🔗

关系分析

  • 人物参与事件关系（participated_in）
  • 事件发生地关系（occurred_at）
  • 人物与人物/事件与事件间的衍生关系
  • 多坐标处理与地理实体连接

快速开始

简单几步，快速部署和使用平台

1

环境准备

系统要求：

  • Python >= 3.9
  • Node.js >= 18
  • Neo4j Desktop 或 Docker
2

安装依赖

# 后端依赖
cd backend
pip install -r requirements.txt

# 前端依赖
cd ../frontend
npm install
3

启动 Neo4j

使用 Neo4j Desktop 或 Docker 启动图数据库服务

配置数据库连接信息（默认：bolt://localhost:7687）

4

启动服务

# 启动后端（端口 8002）
cd backend
python app.py

# 启动前端（端口 3000）
cd frontend
npm start
5

访问平台

打开浏览器访问：http://localhost:3000

开始上传数据、构建图谱、进行可视化分析

历史事件时间轴

重要历史事件的时间线展示

1938年6月

花园口决堤

1938年6月9日，为阻止日军西进，国民党军队在郑州花园口炸开黄河大堤，造成黄河改道，形成大片黄泛区。这一事件对皖北地区产生了深远影响，改变了当地的地理环境和军事格局。

军事事件 地理变迁 皖北地区
1940年

皖北地区军事活动

1940年，皖北地区成为抗日战争的重要战场。多支抗日武装力量在此活动，与日伪军展开激烈战斗。这一时期的军事活动对当地的社会结构和人口分布产生了重大影响。

抗日战争 军事活动
1945年

抗日战争胜利

1945年8月，抗日战争胜利结束。皖北地区经历了长期的战争洗礼，地理环境、人口结构、社会关系都发生了深刻变化。这些变化为后续的军事环境形成奠定了基础。

历史转折 社会变迁
1946-1949年

解放战争时期

解放战争期间，皖北地区再次成为重要战场。淮海战役等重大战役在此展开，大量军事行动和人员流动进一步塑造了当地的军事环境特征。
